# Hollow Knight: Silksong Shader Repo
This repo contains mostly functional shaders for an Asset Ripper project of Hollow Knight: Silksong.  The intention is to git pull this repo into your Asset Ripper project once created to reduce headache with shaders being missing.  If you find one that is still non-functional, either create an (non-duplicate) issue, or if you find the shader and get it working, please upload a PR with the shader in question.
